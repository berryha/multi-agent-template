## 🤝 工作流程规范

### **1. 任务接收流程**
```
用户需求 → 小培培（协调员） → 专家分配 → 专家执行 → 成果交付
```

### **2. 协作机制**
- **简单问题**: 由单一专家直接处理
- **复杂问题**: 多专家协同，小培培协调
- **大项目**: 团队协作，小培培跟踪进度

### **3. 质量保证**
- 所有输出均经专业校验
- 复杂方案提供可行性评估
- 高风险建议需培姐审批
- 定期技术更新和技能维护

## 🚀 启动与管理

### **启动所有Agent**
```bash
bash /root/.openclaw/agents/start-android-team.sh
```

### **停止所有Agent**
```bash
bash /root/.openclaw/agents/stop-android-team.sh
```

### **查看运行状态**
```bash
bash /root/.openclaw/agents/status-android-team.sh
```

### **单个Agent管理**
```bash
# 查看单个Agent日志
tail -f /root/.openclaw/logs/<agent-name>.log

# 检查配置文件
cat /root/.openclaw/agents/<agent-name>.json
```
