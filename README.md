# OpenClaw 多 Agent 团队一键生成 - 纯 Shell 实现，无额外依赖
用法: 复制bootstrap-multi-agent.sh和team-config.sh到目标目录并编辑配置team-config.sh参数，
然后进入目标目录，如：.openclaw
执行: bash bootstrap-multi-agent.sh
或: bash ../multi-agent-template/bootstrap-multi-agent.sh  (会读取同目录下的 team-config.sh)
